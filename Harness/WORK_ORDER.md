# 작업 지시서

## 사용 방법

- 사용자가 반복적으로 AI에게 시킬 작업을 이 문서에 적는다.
- 사용자가 "메모 작업해"라고 말하면 AI는 이 파일을 읽고 작업을 수행한다.
- 현재 대화의 직접 지시가 이 파일보다 우선한다.

## 주의사항

- 이 파일은 템플릿이므로 명시 요청 없이 구조를 지우지 않는다.
- 파일 전체를 비우지 말고, 아래 섹션의 내용만 비워둘 수 있다.
- `유저 메모장`은 작업 대상으로 해석하지 않는다.
- 작업 대상은 `작업 이름`과 `상세 요구사항`에 적힌 내용으로 판단한다.
