# 화면 / UX 하네스

## 시각 방향

- 게임명: SIZE UP
- 장르: 2인 로컬 협동 플랫포머
- 해상도 기준: 가상 해상도 256x144, 창 크기 2560x1440
- 그래픽 방향: 2D 픽셀 그래픽
- 컬러 방향: 단순한 색상 구성
- 아트 목표: 낮은 용량에서도 플레이 정보가 명확히 전달되는 직관적인 실루엣
- UI 톤: 짧은 스테이지와 빠른 재도전에 방해되지 않는 간결한 정보 표시

## UI 방향

- 현재 UI는 커스텀 UI 레이어를 사용한다.
- 텍스트 배치와 자동 크기는 [UI_GUIDE.md](C:/Project/2PGameArcade/UI_GUIDE.md:1)를 따른다.
- UI 텍스트가 화면 밖으로 잘리지 않도록 확인한다.
- 두 플레이어의 상태와 크기 단계가 한눈에 구분되도록 한다.
- 클리어 조건과 재시작 흐름은 짧고 명확하게 표시한다.

## 아트 / 화면 검수 체크리스트

- [ ] 플레이어가 화면 안에 보인다.
- [ ] 픽셀 아트가 흐릿하게 보이지 않는다.
- [ ] UI 텍스트가 잘리지 않는다.
- [ ] 바닥선과 플레이어 위치가 어긋나지 않는다.
- [ ] 2인용 화면 요소가 서로 구분된다.
- [ ] 소형 / 기본 / 대형 상태가 시각적으로 구분된다.
- [ ] 좁은 통로, 높은 벽, 출구 같은 스테이지 목표물이 직관적으로 보인다.
- [ ] 장애물과 발판의 충돌 가능 여부가 헷갈리지 않는다.
- [ ] 에셋 경로 오류가 없다.

## 스타일 메모

- 단순한 색상, 명확한 실루엣, 저용량 에셋을 우선한다.
